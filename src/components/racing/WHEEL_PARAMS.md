# Wheel Physics Parameters

| Parameter | Value | Used In | Description |
|---|---|---|---|
| `WHEEL_RADIUS` | `0.35` | Raycast length calculation, wheel rotation | Wheel radius in world units. Determines raycast origin offset (`radius + suspensionRestLength`), sets the cylinder geometry size, and is used as the denominator when computing wheel rotation speed (`deltaRotation = velocity * delta / radius`). Larger = taller wheel, slower visual spin. |
| `WHEEL_SUSPENSION_REST` | `0.3` | Raycast range, suspension force | Rest length of the suspension spring (distance from chassis connection point to wheel center at rest). Used to compute raycast range (`radius + restLength`) and as the equilibrium point for spring force (`stiffness * (restLength - currentLength)`). Larger = higher ride height, more suspension extension. |
| `WHEEL_SUSPENSION_STIFFNESS` | `30` | Suspension force calculation | Spring constant `k` in the suspension force formula: `force = stiffness * compression * clippedDot`. Higher values make the suspension stiffer (less body roll, harsher over bumps). Lower = softer, more compliant ride but more body lean. |
| `WHEEL_MAX_SUSPENSION_TRAVEL` | `0.3` | Suspension length clamp | Maximum distance the suspension can compress or extend beyond rest length. Clamps suspension to `[rest - travel, rest + travel]`. If suspension extends beyond max, the wheel is considered airborne (no ground contact). Larger = more wheel travel range. |
| `WHEEL_MAX_SUSPENSION_FORCE` | `100000` | Force cap in `applyWheelSuspensionForce` | Upper cap on the suspension impulse applied to the chassis per physics step. Prevents physics explosions from extreme spring forces. High value (100k) effectively uncapped for a 300kg vehicle. |
| `WHEEL_FRICTION_SLIP` | `1` | Friction circle calculation (`maxImp = suspensionForce * delta * frictionSlip`) | Multiplier on the maximum friction impulse the tire can exert. Defines the size of the "friction circle" — the combined limit of forward and lateral forces. Higher = more grip before sliding. Lower = easier to break traction / drift. |
| `WHEEL_DAMPING_RELAXATION` | `2.3` | Suspension force (when extending) | Damping coefficient when the suspension is extending (rebounding). Applied as `force -= dampingRelaxation * velocity` when `velocity >= 0`. Higher = more resistance to suspension extension, less bouncy rebound. |
| `WHEEL_DAMPING_COMPRESSION` | `4.4` | Suspension force (when compressing) | Damping coefficient when the suspension is compressing. Applied as `force -= dampingCompression * velocity` when `velocity < 0`. Higher = more resistance to compression, stiffer reaction to bumps. Currently set higher than relaxation (4.4 vs 2.3) for asymmetric damping. |
| `WHEEL_ROLL_INFLUENCE` | `0.01` | Roll torque in `updateFriction` | Scales the vertical (up-axis) component of the contact point offset when applying lateral friction impulse. Controls how much lateral forces cause body roll. `0.01` nearly eliminates roll — the lateral impulse is applied almost at the chassis center vertically. Higher = more body roll in corners. |
| `WHEEL_SIDE_FRICTION` | `2` | Side impulse scaling (`sideImpulse *= sideFrictionStiffness`) | Multiplier on the resolved lateral friction impulse. Controls how strongly the tire resists sideways sliding. Higher = more lateral grip, sharper cornering. Lower = more oversteer/drifting tendency. |
| `WHEEL_FORWARD_ACCEL` | `2` | Friction ellipse normalization (`x = fwdImpulse / fwdFactor / forwardAccel`) | Normalizes forward impulse in the combined friction ellipse equation (`x² + y²`). Effectively controls how much of the friction budget is allocated to forward forces. Lower = forward forces saturate the friction circle sooner, limiting acceleration. Higher = more forward grip available. |
| `WHEEL_SIDE_ACCEL` | `2` | Friction ellipse normalization (`y = sideImpulse / sideFactor / sideAccel`) | Normalizes lateral impulse in the combined friction ellipse. Controls friction budget allocation for sideways forces. Lower = lateral forces saturate sooner (easier to slide sideways). Higher = more lateral grip budget. |
| `WHEEL_CUSTOM_SLIDING_ROT_SPEED` | `-30` | Wheel visual rotation when sliding | When the wheel is sliding (tire broken free) and engine force is applied, this overrides the physics-based wheel rotation with a fixed angular speed. Negative = correct spin direction. Higher magnitude = faster wheel spin visual during burnouts/slides. |
